# When Kansas Bleeds

A slightly overdue entry for the 1995 IFComp. Maybe I can have it for the 120th anniversary of the battle?

## Modules

`kansas.dg` -- main game file

`grid.dg` -- navigation on the open prairie

`time.dg` -- handling of date and time

`d6.dg` -- UNDO-safe random numbers

`d6-lite.dg` -- randomness that uses the builtin RNG

`utils.dg` -- useful utility predicates

`unit.dg` -- unit test runner

`stdlib.dg` -- Dialog standard library

`stddebug.dg` -- helpful debugging predicates

`Makefile` -- builds and tests the game
